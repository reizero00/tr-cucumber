selenium-cucumber : Automation Tesing Using Ruby

This framework was built off of github.com/selenium-cucumber/selenium-cucumber-ruby follow the instructions on that repo for installation

Alternate steps:

  1. Install ruby

  2. gem install cucumber selenium-webdriver chunky_png rspec appium_lib headless chromedriver-helper capybara capybara-webkit capybara-screenshot

Ubuntu Installation

Ubuntu 14 and above require additional dependencies.

Install Ruby 2.3

  1. sudo apt-add-repository ppa:brightbox/ruby-ng

  2. sudo apt-get update

  3. sudo apt-get install ruby2.3 ruby2.3-dev

Install Capybara

  1. sudo apt-get install libqt4-dev libqtwebkit-dev

  2. sudo apt-get install build-essential patch

  3. sudo apt-get install ruby-dev zlib1g-dev liblzma-dev

  4. sudo gem install nokogiri

  5. sudo gem install capybara -v 2.7.1

Install the rest of the stuff

  1. sudo gem install cucumber chunky_png rspec appium_lib headless chromedriver-helper typhoeus

  2. sudo gem install selenium-webdriver -v 2.53.4

How to run tests

Prerequisites - Selenium Cucumber Ruby is installed - Tests have been downloaded

  1. In the terminal navigate to cd ~/liferay-qa-ee/testray

  2. Run tests by executing cucumber

  3. This will run all tests in the features folder in alphabetical order

  4. To run a specific feature file you can run cucumber features/projects.feature
